## CREATE-ROLE

### Name

CREATE ROLE

### Description

The CREATE ROLE statement is used to create a role.

```sql
  CREATE ROLE rol_name;
````

This statement creates a role with no specified permissions. You can grant permissions to the role by the GRANT command.

### Example

1. Create a role

    ```sql
    CREATE ROLE role1;
    ````
